Задание
```
Lab 14 (till 18 december)

1. Write a script 'Lab14-1' that prints the number of users that have
   home directory in directory '/home/'.
2. Write a script 'Lab14-2' that prints the number of users that starts
   '/bin/bash' after logs into system; or the number of pseudo-users
   (that have 'nologin' in that field). The contents of output is
   specified by first argument. Thus, argument should be 'bash' or
   'nologin'.
3. Write a script 'Lab14-3' that prints the User name field and
   Gecos field sorted by username, is first argument is 'user', or
   by Gecos, if first argument is 'gecos'.

```
Порядок выполнения команд:

```
./Lab14-1
./Lab14-2 bash
./Lab14-2 nologin
./Lab14-3 user
./Lab14-3 gecos

```
