Задание:
```
Lab 16 (till 25 december)

1. Write a script 'Lab16-1' that displays all network interfaces of your system
   in the form: "<if>: <IP4 address>", for example, "lo: 127.0.0.1".
2. Write a script 'Lab16-2' that displays gateway for your main interface.
   For example, "wlo1: 10.10.10.1 ".
```
Порядок выполнения команд:

```
./Lab16-1
./Lab16-2
```
